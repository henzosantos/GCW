Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de dar visibilidade e apresentar números da carreira de um dos jogadores mais midiáticos do Brasil.
A aplicação busca oferecer uma solução simples, moderna e eficiente, permitindo que o máximo de pessoas possa conhecer e acompanhar a trajetória do grande jogador Neymar Jr.
---
Funcionalidades

✔️ Home
✔️ Carreira
✔️ Trajetoria
✔️ historia
